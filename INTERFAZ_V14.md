# Interfaz v14 — Tablero compacto para tablet

## Cambios principales

- El rol `rebanado` entra directamente a `/vales/tablero` después de iniciar sesión.
- Conserva acceso a Inicio, Pantalla almacén y Cierre de día según sus permisos.
- El tablero usa un fondo institucional café/vino oscuro.
- Los vales se muestran como tarjetas claras y compactas.
- En tablet horizontal se muestran dos tarjetas por fila; en escritorio amplio, hasta tres.
- En tablet vertical y celular se adapta a una sola columna.
- El estado conserva una franja de color, distintivo y flujo visual.
- Los botones siguen siendo táctiles, con altura suficiente para operación.
