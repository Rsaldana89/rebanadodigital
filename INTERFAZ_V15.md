# Interfaz V15 — Modo operativo simplificado

Esta versión conserva la identidad institucional vino, crema y café, pero reduce la carga visual del tablero para el rol **Rebanado**.

## Cambios principales

- Encabezado operativo compacto, sin texto explicativo redundante.
- Indicadores de activos y atrasados integrados en una sola línea.
- Fecha, búsqueda y filtros reunidos en un único bloque.
- Filtros por estado convertidos en botones compactos.
- Se eliminó el flujo gráfico repetido dentro de cada vale para Rebanado.
- Datos del vale agrupados en líneas simples, sin exceso de barras o tarjetas.
- Una sola acción principal visible según el estado actual.
- Cancelar se movió a un menú discreto de acciones adicionales.
- El estado Listo se presenta como “Listo para entrega”, sin mostrar un botón Entregar deshabilitado.
- CEDIS y Administrador conservan el tablero completo anterior.
- El mensaje de bienvenida no ocupa espacio en el tablero de Rebanado; otros mensajes operativos se muestran brevemente.
- `.npmrc` ya no excluye dependencias de desarrollo, por lo que `npm run dev` vuelve a instalar y reconocer `nodemon`.
