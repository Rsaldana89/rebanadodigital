# Interfaz v9

## Cambios principales

- Se agregó un diagrama informativo del flujo en la pantalla de inicio de sesión.
- Se eliminaron las tarjetas explicativas anteriores del login.
- Se agregó un botón con icono de ojo para mostrar u ocultar la contraseña.
- Se agregó la opción para recordar usuario y contraseña en el navegador actual.
- Cada vale muestra un minidiagrama del proceso:
  - Pendiente
  - Rebanando
  - Listo
  - Entregado
- El paso actual se resalta en vino institucional y los pasos concluidos aparecen en verde.
- El estado Cancelado no se presenta como una etapa del flujo.
